# 2026-06-25 Phase 5：小程序 P1/P2 增强（M3-M10）

**类型**：功能发布
**影响范围**：小程序端、设备网关、任务系统

## 新增能力

- **任务模板**：常用任务一键下发，支持自定义参数。
- **微信订阅通知**：任务完成、设备离线等事件通过微信订阅消息触达。
- **素材库**：图片/SVG/音频素材的统一上传、管理与复用。
- **批量绘图**：一次选择多个素材或参数，批量生成创作结果。
- **创作结果预览**：SVG 渲染、保存相册、分享到微信好友/朋友圈。
- **设备分享**：生成分享二维码，支持访客模式与权限管理。
- **设备发现**：蓝牙/Wi-Fi 辅助配网，降低首次绑定门槛。
- **统计分析**：用量、任务成功率、设备活跃度可视化。

## 修复与优化

- 修复 Phase 5 全量 pytest 中的失败用例，测试全部通过。
- 增强设备远程证明与权限控制，区分 view / control 分享权限。
- 微信通知订阅校验 `deviceIds`，空列表不再匹配所有设备。
