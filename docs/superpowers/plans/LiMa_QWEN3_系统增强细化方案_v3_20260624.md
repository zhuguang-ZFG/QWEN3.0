### 1.2 保留的模块

以下模块虽涉及代码检测，但功能不限于编码，**予以保留**：

| 模块 | 保留原因 |
|---|---|
| `routing_classifier.py` 的 `classify()` | IDE 检测（`return "ide"`）服务于所有 IDE 场景，非编码专有 |
| `speculative.py` facade | 投机执行本身是通用能力，仅移除 `"code"` 亲和池 |
| `speculative_execution.py` | 并行投机执行引擎，与编码无关 |
| `backends_constants_code_tools.py` 的 `TOOL_CAPABLE_BACKENDS` | Tool calling 是通用能力 |
| `context_pipeline/routing_bridge.py` | 路由桥接，非编码专有 |

### 1.3 退役影响评估

- **路由影响**：移除 coding 场景后，原 coding 请求将走 standard execute 路径，不再有代码后端优先选择和质量重试
- **性能影响**：减少 4 个上下文注入步骤（代码扫描/语义检索/图谱/记忆），减少 ~50-200ms 延迟
- **token 影响**：不再注入代码上下文 system message（原 4000 字符上限），节省 ~1000 token
- **测试影响**：需移除所有 coding 相关测试用例，预计减少 ~200-300 个测试

**现状**：`canary.py` 仅支持手动添加金丝雀设备，无自动灰度推进。

```python
"""Gradual rollout engine — auto-progress through 5% → 20% → 50% → 100%."""

class GradualRollout:
    STAGES = [
        ("canary", 0.05),    # 5% 设备
        ("early", 0.20),     # 20% 设备
        ("mid", 0.50),       # 50% 设备
        ("full", 1.00),      # 100% 设备
    ]

    def __init__(self, state_path):
        self._state_path = state_path
        self._stage_index = 0
        self._stage_success = 0
        self._stage_failure = 0
        self._auto_promote = True
        self._promote_threshold = 0.95  # 成功率阈值
        self._min_samples = 10           # 最少样本数

    def select_devices_for_stage(self, all_devices: list[str], stage_ratio: float) -> list[str]:
        """按比例选择设备，优先选择活跃度低的设备减少风险。"""
        import hashlib
        tagged = [(d, int(hashlib.md5(d.encode()).hexdigest(), 16) % 100) for d in all_devices]
        tagged.sort(key=lambda x: x[1])
        cutoff = int(len(tagged) * stage_ratio)
        return [d for d, _ in tagged[:cutoff]]

    def should_promote(self) -> bool:
        total = self._stage_success + self._stage_failure
        if total < self._min_samples:
            return False
        return (self._stage_success / total) >= self._promote_threshold

    def should_rollback(self) -> bool:
        total = self._stage_success + self._stage_failure
        if total < self._min_samples:
            return False
        return (self._stage_failure / total) > 0.15  # 失败率 > 15%

    def promote(self):
        if self._stage_index < len(self.STAGES) - 1:
            self._stage_index += 1
            self._stage_success = 0
            self._stage_failure = 0
            self._save()

    def rollback(self):
        self._stage_index = max(0, self._stage_index - 1)
        self._stage_success = 0
        self._stage_failure = 0
        self._save()
```

```python
"""Auto-rollback monitor — background thread checking canary health."""

class RollbackMonitor:
    """每 60 秒检查金丝雀健康，连续 3 次不健康自动回滚。"""

    def __init__(self, gradual: GradualRollout, canary: CanaryDeployment):
        self._gradual = gradual
        self._canary = canary
        self._unhealthy_count = 0
        self._threshold = 3
        self._interval = 60

    def check_and_act(self) -> str:
        """检查健康状态，返回 'healthy' / 'unhealthy' / 'rolled_back'。"""
        if self._canary.is_healthy():
            self._unhealthy_count = 0
            if self._gradual.should_promote():
                self._gradual.promote()
                return "promoted"
            return "healthy"
        else:
            self._unhealthy_count += 1
            if self._unhealthy_count >= self._threshold:
                self._gradual.rollback()
                return "rolled_back"
            return "unhealthy"
```

#### F1-3 固件签名验证增强

**现状**：`device_ota.py` 的 `_firmware_metadata()` 仅校验 SHA256 和签名存在性，不验证签名内容。

**方案**：新建 `device_ota/signature.py`

```python
"""Firmware signature verification using Ed25519."""

from cryptography.hazmat.primitives.asymmetric.ed25519 import Ed25519PublicKey
from cryptography.hazmat.primitives import serialization
import base64, hashlib

class FirmwareVerifier:
    def __init__(self, public_key_pem: str):
        self._public_key = serialization.load_pem_public_key(public_key_pem.encode())

    def verify(self, firmware_url: str, firmware_sha256: str, signature: str) -> bool:
        """验证固件签名：SHA256(firmware_url + sha256) 用 Ed25519 私钥签名。"""
        message = (firmware_url + firmware_sha256).encode()
        sig_bytes = base64.b64decode(signature)
        try:
            self._public_key.verify(sig_bytes, message)
            return True
```

#### F3-1 路径优化器

```python
"""Path optimizer — compress, smooth, and reorder motion paths."""

class PathOptimizer:
    def compress(self, path: list[dict], tolerance: float = 0.5) -> list[dict]:
        """合并连续共线点，减少路径点数。Douglas-Peucker 简化。"""
        if len(path) < 3:
            return path
        result = [path[0]]
        for i in range(1, len(path) - 1):
            prev, curr, next_pt = path[i-1], path[i], path[i+1]
            # 计算点到线段的距离
            dist = self._point_to_line_distance(curr, prev, next_pt)
            if dist > tolerance:
                result.append(curr)
        result.append(path[-1])
        return result

    def smooth(self, path: list[dict], iterations: int = 2) -> list[dict]:
        """B-spline 平滑，减少拐角锐度。"""
        for _ in range(iterations):
            new_path = [path[0]]
            for i in range(1, len(path) - 1):
                new_x = 0.25 * path[i-1]["x"] + 0.5 * path[i]["x"] + 0.25 * path[i+1]["x"]
                new_y = 0.25 * path[i-1]["y"] + 0.5 * path[i]["y"] + 0.25 * path[i+1]["y"]
                new_path.append({"x": round(new_x, 2), "y": round(new_y, 2), "z": path[i]["z"]})
            new_path.append(path[-1])
            path = new_path
        return path

    def optimize_travel(self, path: list[dict]) -> list[dict]:
        """重排序 pen-up 段以减少空行程（最近邻贪心）。"""
        segments = self._split_by_pen_up(path)
        if len(segments) <= 1:
            return path
        ordered = [segments[0]]
        remaining = segments[1:]
        while remaining:
            last_end = ordered[-1][-1]
            nearest = min(remaining, key=lambda seg: self._dist(last_end, seg[0]))
            ordered.append(nearest)
            remaining.remove(nearest)
        return [pt for seg in ordered for pt in seg]

    def _point_to_line_distance(self, p, a, b) -> float: ...
    def _split_by_pen_up(self, path) -> list[list[dict]]: ...
    def _dist(self, a, b) -> float: ...
```

#### F5-1 证明验证器

```python
# device_gateway/attestation.py
"""Firmware remote attestation — verify device integrity on connect.

流程:
  1. 设备连接时上报 firmware_version + firmware_hash（SHA-256）
  2. 服务器比对已知固件哈希白名单
  3. 通过 → full_access；不匹配 → read_only；未知版本 → quarantine
"""

from __future__ import annotations

import hashlib
import logging
from dataclasses import dataclass
from typing import Any

_log = logging.getLogger(__name__)

# 固件哈希白名单（从 config/firmware_hashes.json 加载，支持热更新）
# 格式: {"v1.0.0": "sha256:abcdef...", "v1.1.0": "sha256:123456..."}
_KNOWN_HASHES: dict[str, str] = {}

# 降级策略
ACTION_FULL_ACCESS = "full_access"
ACTION_READ_ONLY = "read_only"       # 只能接收任务，不能发送指令
ACTION_QUARANTINE = "quarantine"     # 完全隔离，仅允许 OTA 升级


@dataclass(frozen=True)
class AttestationResult:
    verified: bool
    action: str
    reason: str
    firmware_version: str
    expected_hash: str | None
    received_hash: str


class AttestationVerifier:
    """固件远程证明验证器。"""

    def __init__(self, hashes: dict[str, str] | None = None) -> None:
        if hashes:
            self._hashes = dict(hashes)
        else:
            self._hashes = dict(_KNOWN_HASHES)

    def reload_hashes(self, path: str = "config/firmware_hashes.json") -> None:
        """从 JSON 文件重新加载固件哈希白名单。"""
        import json
        from pathlib import Path
        p = Path(path)
        if p.exists():
            with open(p, encoding="utf-8") as f:
                self._hashes = json.load(f)
            _log.info("attestation: loaded %d firmware hashes from %s", len(self._hashes), path)

    def verify(self, device_id: str, firmware_hash: str, firmware_version: str) -> AttestationResult:
        """验证设备固件完整性。

        Args:
            device_id: 设备唯一标识
            firmware_hash: 设备上报的固件 SHA-256 哈希（hex 或 sha256: 前缀）
            firmware_version: 固件版本号（如 "v1.2.0"）

        Returns:
            AttestationResult: 验证结果和降级动作
        """
        normalized_hash = firmware_hash.removeprefix("sha256:").lower()
        expected = self._hashes.get(firmware_version)

        if expected is None:
            _log.warning("attestation: unknown firmware version %s for device %s", firmware_version, device_id)
            return AttestationResult(
                verified=False, action=ACTION_QUARANTINE,
                reason="unknown_firmware_version",
                firmware_version=firmware_version,
                expected_hash=None, received_hash=normalized_hash,
            )

        expected_normalized = expected.removeprefix("sha256:").lower()
        if normalized_hash != expected_normalized:
            _log.warning("attestation: hash mismatch for device %s (version=%s)", device_id, firmware_version)
            return AttestationResult(
                verified=False, action=ACTION_READ_ONLY,
                reason="hash_mismatch",
                firmware_version=firmware_version,
                expected_hash=expected_normalized, received_hash=normalized_hash,
            )

        _log.info("attestation: device %s verified (version=%s)", device_id, firmware_version)
        return AttestationResult(
            verified=True, action=ACTION_FULL_ACCESS,
            reason="ok",
            firmware_version=firmware_version,
            expected_hash=expected_normalized, received_hash=normalized_hash,
        )

    def compute_firmware_hash(self, firmware_bytes: bytes) -> str:
        """计算固件二进制的 SHA-256 哈希（用于生成白名单）。"""
        return hashlib.sha256(firmware_bytes).hexdigest()


# 全局单例
verifier = AttestationVerifier()
```

```c
// 设备端：计算固件分区 SHA-256
#include <esp_partition.h>
#include <mbedtls/sha256.h>

void compute_firmware_hash(char *hash_hex_out, size_t out_len) {
    const esp_partition_t *running = esp_ota_get_running_partition();
    mbedtls_sha256_context ctx;
    mbedtls_sha256_init(&ctx);
    mbedtls_sha256_starts(&ctx, 0);

    // 分块读取固件分区并更新哈希
    uint8_t buf[4096];
    for (size_t offset = 0; offset < running->size; offset += sizeof(buf)) {
        size_t chunk = sizeof(buf);
        if (offset + chunk > running->size) chunk = running->size - offset;
        esp_partition_read(running, offset, buf, chunk);
        mbedtls_sha256_update(&ctx, buf, chunk);
    }

    uint8_t hash[32];
    mbedtls_sha256_finish(&ctx, hash);
    mbedtls_sha256_free(&ctx);

    // 转 hex 字符串
    for (int i = 0; i < 32 && i * 2 + 1 < out_len; i++) {
        snprintf(hash_hex_out + i * 2, 3, "%02x", hash[i]);
    }
}
```

```python
"""Multi-device coordination — split large artwork across devices."""

class MultiDeviceCoordinator:
    def split_artwork(self, svg_content: str, device_count: int) -> list[dict]:
        """将 SVG 按区域分割给多台设备。"""
        from device_gateway.svg_parser import parse_svg_bounds
        bounds = parse_svg_bounds(svg_content)
        regions = self._grid_split(bounds, device_count)
        return [
            {"device_index": i, "region": r, "svg_clip": self._clip_svg(svg_content, r)}
            for i, r in enumerate(regions)
        ]

    def merge_results(self, device_results: list[dict]) -> dict:
        """合并多设备执行结果。"""
        return {
            "total_devices": len(device_results),
            "success_count": sum(1 for r in device_results if r["status"] == "completed"),
            "failed_count": sum(1 for r in device_results if r["status"] == "failed"),
            "overall_status": "completed" if all(r["status"] == "completed" for r in device_results) else "partial",
        }

    def _grid_split(self, bounds: dict, n: int) -> list[dict]:
        """将边界框按网格分割为 n 个区域。

        Args:
            bounds: {"x": 0, "y": 0, "width": 200, "height": 200}
            n: 设备数量

        Returns:
            [{"x": 0, "y": 0, "width": 100, "height": 100}, ...]
        """
        import math
        cols = math.ceil(math.sqrt(n))
        rows = math.ceil(n / cols)
        cell_w = bounds["width"] / cols
        cell_h = bounds["height"] / rows
        regions = []
        for r in range(rows):
            for c in range(cols):
                if len(regions) >= n:
                    break
                regions.append({
                    "x": bounds["x"] + c * cell_w,
                    "y": bounds["y"] + r * cell_h,
                    "width": cell_w,
                    "height": cell_h,
                })
        return regions

    def _clip_svg(self, svg_content: str, region: dict) -> str:
        """用 clipPath 裁剪 SVG 到指定区域。"""
        import re
        clip_id = f"clip_{region['x']}_{region['y']}"
        clip_def = (
            f'<defs><clipPath id="{clip_id}">'
            f'<rect x="{region["x"]}" y="{region["y"]}" '
            f'width="{region["width"]}" height="{region["height"]}"/>'
            f'</clipPath></defs>'
        )
        # 在 <svg> 标签后插入 clipPath 定义，并在根 <g> 上添加 clip-path 属性
        svg_with_clip = re.sub(
            r'(<svg[^>]*>)',
            rf'\1{clip_def}',
            svg_content,
            count=1,
        )
        # 在第一个 <g> 标签上添加 clip-path
        svg_with_clip = re.sub(
            r'<g(?![^>]*clip-path)',
            f'<g clip-path="url(#{clip_id})"',
            svg_with_clip,
            count=1,
        )
        return svg_with_clip

    def assign_devices(self, regions: list[dict], device_ids: list[str]) -> list[dict]:
        """将区域分配给具体设备。"""
        return [
            {**region, "device_id": device_ids[i], "device_index": i}
            for i, region in enumerate(regions[:len(device_ids)])
        ]

    async def execute_coordinated(
        self, svg_content: str, device_ids: list[str], coordinator_id: str
    ) -> dict:
        """协调整个多设备绘制流程。"""
        from device_gateway.tasks import enqueue_task
        from device_gateway.sessions import registry

        # 1. 分割图案
        regions = self._grid_split(parse_svg_bounds(svg_content), len(device_ids))
        assignments = self.assign_devices(regions, device_ids)

        # 2. 为每台设备生成裁剪后的 SVG 任务
        batch_id = new_id()
        results = []
        for assignment in assignments:
            clipped_svg = self._clip_svg(svg_content, assignment)
            session = registry.get(assignment["device_id"])
            if not session:
                results.append({
                    "device_id": assignment["device_id"],
                    "status": "failed",
                    "error": "device_offline",
                })
                continue

            task = {
                "task_id": new_id(),
                "batch_id": batch_id,
                "capability": "draw_svg",
                "params": {"svg": clipped_svg, "region": assignment},
            }
            await enqueue_task(assignment["device_id"], task)
            results.append({
                "device_id": assignment["device_id"],
                "task_id": task["task_id"],
                "status": "dispatched",
            })

        # 3. 等待所有设备完成（或超时）
        # 4. 合并结果
        return {
            "batch_id": batch_id,
            "coordinator_id": coordinator_id,
            "results": results,
            "summary": self.merge_results(results),
        }
```

```python
# device_ledger/projection.py — 新建
"""Event projection — rebuild task state from event stream.

投影器从事件流重建当前状态，支持:
- 任务状态重建（status / progress / error）
- 任务时间线查询
- 设备活动摘要
- 任务耗时统计
"""

from __future__ import annotations

from collections import defaultdict
from typing import Any

from device_ledger.events import LedgerEvent
from device_ledger.store import ledger_store


class TaskProjection:
    """任务状态投影器 — 从事件流重建任务当前状态。"""

    # 状态转换矩阵: (当前状态, 事件类型) → 新状态
    _TRANSITIONS = {
        ("unknown", "task_created"): "created",
        ("created", "task_dispatched"): "dispatched",
        ("dispatched", "task_acknowledged"): "executing",
        ("executing", "task_progress"): "executing",
        ("executing", "task_paused"): "paused",
        ("paused", "task_resumed"): "executing",
        ("executing", "task_terminal"): "completed",
        ("executing", "task_terminal"): "failed",
        ("dispatched", "task_terminal"): "failed",
        ("created", "task_terminal"): "cancelled",
        ("paused", "task_terminal"): "cancelled",
    }

    def rebuild_state(self, task_id: str) -> dict[str, Any]:
        """从事件流重建任务当前状态。"""
        events = ledger_store.events_for_task(task_id)
        if not events:
            return {"task_id": task_id, "status": "unknown", "events": 0}

        state: dict[str, Any] = {
            "task_id": task_id,
            "status": "unknown",
            "device_id": None,
            "created_at": None,
            "progress": 0,
            "error": None,
            "event_count": len(events),
            "last_event_at": events[-1].created_at if events else None,
        }

        for event in events:
            if event.event_type == "task_created":
                state["status"] = "created"
                state["device_id"] = event.device_id
                state["created_at"] = event.created_at
                state.update(event.payload)
            elif event.event_type == "task_dispatched":
                state["status"] = "dispatched"
                state["dispatched_at"] = event.created_at
            elif event.event_type == "task_acknowledged":
                state["status"] = "executing"
                state["acknowledged_at"] = event.created_at
            elif event.event_type == "task_progress":
                state["progress"] = event.payload.get("progress", state["progress"])
            elif event.event_type == "task_paused":
                state["status"] = "paused"
            elif event.event_type == "task_resumed":
                state["status"] = "executing"
            elif event.event_type == "task_terminal":
                terminal_status = event.payload.get("terminal_status", "failed")
                state["status"] = terminal_status  # completed / failed / cancelled
                state["error"] = event.payload.get("error")
                state["terminal_at"] = event.created_at
                state["duration_ms"] = event.payload.get("duration_ms")
            elif event.event_type == "motion_event":
                state.setdefault("motion_events", []).append(event.payload)

        return state

    def timeline(self, task_id: str) -> list[dict[str, Any]]:
        """返回任务完整时间线（按时间排序）。"""
        events = ledger_store.events_for_task(task_id)
        return [
            {
                "event_id": e.event_id,
                "event_type": e.event_type,
                "task_id": e.task_id,
                "device_id": e.device_id,
                "payload": e.payload,
                "created_at": e.created_at,
            }
            for e in events
        ]

    def task_duration(self, task_id: str) -> dict[str, Any] | None:
        """计算任务耗时明细。"""
        events = ledger_store.events_for_task(task_id)
        if len(events) < 2:
            return None

        created = next((e for e in events if e.event_type == "task_created"), None)
        terminal = next((e for e in reversed(events) if e.event_type == "task_terminal"), None)
        if not created or not terminal:
            return None

        from datetime import datetime
        start = datetime.fromisoformat(created.created_at.replace("Z", "+00:00"))
        end = datetime.fromisoformat(terminal.created_at.replace("Z", "+00:00"))
        total_ms = int((end - start).total_seconds() * 1000)

        # 计算各阶段耗时
        dispatched = next((e for e in events if e.event_type == "task_dispatched"), None)
        acknowledged = next((e for e in events if e.event_type == "task_acknowledged"), None)

        return {
            "task_id": task_id,
            "total_ms": total_ms,
            "queue_ms": int((datetime.fromisoformat(dispatched.created_at.replace("Z", "+00:00")) - start).total_seconds() * 1000) if dispatched else None,
            "dispatch_ms": int((datetime.fromisoformat(acknowledged.created_at.replace("Z", "+00:00")) - datetime.fromisoformat(dispatched.created_at.replace("Z", "+00:00"))).total_seconds() * 1000) if dispatched and acknowledged else None,
            "execute_ms": int((end - datetime.fromisoformat(acknowledged.created_at.replace("Z", "+00:00"))).total_seconds() * 1000) if acknowledged else None,
        }


class DeviceProjection:
    """设备活动投影器 — 从事件流重建设备活动摘要。"""

    def device_summary(self, device_id: str, limit: int = 100) -> dict[str, Any]:
        """设备活动摘要。"""
        events = ledger_store.events_for_device(device_id)
        if not events:
            return {"device_id": device_id, "total_events": 0}

        task_ids = set(e.task_id for e in events if e.task_id)
        terminal_events = [e for e in events if e.event_type == "task_terminal"]
        completed = sum(1 for e in terminal_events if e.payload.get("terminal_status") == "completed")
        failed = sum(1 for e in terminal_events if e.payload.get("terminal_status") == "failed")

        return {
            "device_id": device_id,
            "total_events": len(events),
            "unique_tasks": len(task_ids),
            "completed_tasks": completed,
            "failed_tasks": failed,
            "success_rate": round(completed / max(len(terminal_events), 1) * 100, 1),
            "last_activity": events[-1].created_at if events else None,
        }
```

```python
# routes/device_app_misc.py — 新增

@router.websocket("/devices/{device_id}/ws")
async def device_status_ws(websocket: WebSocket, device_id: str,
                           authorization: str = Query(default="")):
    """小程序 WebSocket — 实时推送设备状态变更。

    事件类型:
      - device_online: 设备上线
      - device_offline: 设备离线
      - task_started: 任务开始执行
      - task_completed: 任务完成
      - task_failed: 任务失败
      - task_progress: 任务进度更新
      - firmware_update: 固件更新可用
    """
    from access_guard import authenticate_websocket, configured_api_keys
    from device_gateway.sessions import registry

    # 认证
    authorized, _ = authenticate_websocket(websocket, authorization)
    if not authorized:
        if not configured_api_keys():
            await websocket.close(code=1011, reason="API key not configured")
            return
        await websocket.close(code=4401, reason="Unauthorized")
        return

    await websocket.accept()

    # 发送当前状态
    session = registry.get(device_id)
    await websocket.send_json({
        "type": "status_snapshot",
        "deviceId": device_id,
        "online": session is not None,
        "working": bool(session and session.inflight_tasks) if session else False,
        "firmwareVersion": session.fw_rev if session else None,
        "capabilities": session.capabilities if session else [],
        "lastUptimeMs": session.last_uptime_ms if session else 0,
    })

    # 轮询设备状态变更（10秒间隔）
    import asyncio
    last_online = session is not None
    last_working = False
    last_task_id = None

    try:
        while True:
            await asyncio.sleep(10)  # 10s 轮询间隔
            session = registry.get(device_id)
            online = session is not None
            working = bool(session and session.inflight_tasks) if session else False
            current_task_id = None

            if session and session.inflight_tasks:
                with session.inflight_lock:
                    tasks = list(session.inflight_tasks.values())
                    if tasks:
                        current_task_id = tasks[0].get("task_id")

            # 检测状态变更
            if online != last_online:
                await websocket.send_json({
                    "type": "device_online" if online else "device_offline",
                    "deviceId": device_id,
                    "timestamp": now(),
                })
                last_online = online

            if working != last_working:
                event_type = "task_started" if working else "task_completed"
                await websocket.send_json({
                    "type": event_type,
                    "deviceId": device_id,
                    "taskId": current_task_id or last_task_id,
                    "timestamp": now(),
                })
                last_working = working

            if current_task_id != last_task_id and current_task_id:
                last_task_id = current_task_id

    except WebSocketDisconnect:
        pass
    except Exception as exc:
        _log.warning("device_status_ws error for device %s: %s", device_id, exc)
    finally:
        try:
            await websocket.close()
        except Exception:
            pass
```

```python
# 微信API配置（从环境变量读取）
_WX_APPID = ""  # LIMA_WX_APPID
_WX_SECRET = ""  # LIMA_WX_SECRET
_WX_ACCESS_TOKEN_URL = "https://api.weixin.qq.com/cgi-bin/token"
_WX_SUBSCRIBE_MSG_URL = "https://api.weixin.qq.com/cgi-bin/message/subscribe/send"


class WeChatNotifier:
    """微信订阅消息推送器。"""

    def __init__(self) -> None:
        self._access_token: str = ""
        self._token_expires: float = 0

    def _get_access_token(self) -> str:
        """获取微信 access_token（带缓存）。"""
        import time
        if self._access_token and time.time() < self._token_expires - 60:
            return self._access_token
        resp = httpx.get(_WX_ACCESS_TOKEN_URL, params={
            "grant_type": "client_credential",
            "appid": _WX_APPID,
            "secret": _WX_SECRET,
        }, timeout=10)
        data = resp.json()
        if "access_token" not in data:
            _log.error("wx access_token failed: %s", data)
            return ""
        self._access_token = data["access_token"]
        self._token_expires = time.time() + data.get("expires_in", 7200)
        return self._access_token

    async def send_subscribe_message(
        self, openid: str, template_key: str, data: dict[str, str], page: str = ""
    ) -> dict[str, Any]:
        """发送微信订阅消息。"""
        tmpl = NOTIFICATION_TEMPLATES.get(template_key)
        if not tmpl or not tmpl["template_id"]:
            _log.warning("notification template '%s' not configured", template_key)
            return {"status": "failed", "error": "template_not_configured"}

        token = self._get_access_token()
        if not token:
            return {"status": "failed", "error": "no_access_token"}

        # 构建消息体
        wx_data = {}
        for wx_key, data_key in tmpl["data_keys"].items():
            wx_data[wx_key] = {"value": data.get(data_key, "")}

        payload = {
            "touser": openid,
            "template_id": tmpl["template_id"],
            "page": page or tmpl["page"].format(**data),
            "data": wx_data,
            "miniprogram_state": "formal",  # formal / trial / developer
        }

        try:
            resp = httpx.post(_WX_SUBSCRIBE_MSG_URL, params={"access_token": token}, json=payload, timeout=10)
            result = resp.json()
            if result.get("errcode", 0) != 0:
                _log.error("wx subscribe msg failed: %s", result)
            return {"status": "sent" if result.get("errcode", 0) == 0 else "failed", "wx_response": result}
        except Exception as exc:
            _log.error("wx subscribe msg error: %s", exc)
            return {"status": "failed", "error": str(exc)}

async def dispatch_notification(device_id: str, event_type: str, data: dict[str, str]) -> None:
    """设备状态变更时推送通知给所有订阅者。

    在 device_gateway/tasks.py 的任务状态变更回调中调用。
    """
    from device_logic.db import connect
    from device_logic.notifications import notifier

    tmpl = NOTIFICATION_TEMPLATES.get(event_type)
    if not tmpl:
        return

    # 查询订阅了该设备+该事件类型的用户
    with connect() as conn:
        rows = conn.execute(
            """SELECT s.openid, s.account_id FROM v2_notification_subscription s
               WHERE s.status = 'active'
               AND (s.device_ids IS NULL OR s.device_ids LIKE ?)
               AND s.template_ids LIKE ?""",
            (f'%"{device_id}"%', f'%"{event_type}"%')
        ).fetchall()

    if not rows:
        return

    for row in rows:
        result = await notifier.send_subscribe_message(
            openid=row["openid"],
            template_key=event_type,
            data=data,
        )
        # 记录发送日志
        with connect() as conn:
            conn.execute(
                """INSERT INTO v2_notification_log
                   (id, account_id, device_id, event_type, template_id, payload, sent_at, status, wx_response)
                   VALUES (?,?,?,?,?,?,?,?,?)""",
                (new_id(), row["account_id"], device_id, event_type,
                 tmpl["template_id"], json.dumps(data, ensure_ascii=False),
```

```python
    此端点也可被管理后台调用，通过服务器端 mDNS 扫描。
    """
    account = authorize(authorization)
    if isinstance(account, JSONResponse):
        return account
    body = await read_body(request)
    # 方式1: 小程序直接扫描，将结果上报
    if body and body.get("devices"):
        # 小程序已通过 UDP 发现设备，上报到服务器验证是否未绑定
        discovered = body["devices"]
        with connect() as conn:
            bound_sns = {
                row["device_sn"] for row in conn.execute(
                    "SELECT device_sn FROM v2_device_binding WHERE status != 'unbound'"
                ).fetchall()
            }
        result = [
            {**d, "bound": d.get("deviceSn", "") in bound_sns}
            for d in discovered
        ]
        return ok({"devices": result, "source": "client_report"})

    # 方式2: 服务器端 mDNS 扫描（可选，需要服务器在同一局域网）
    try:
        import socket
        import json as json_mod
        # 发送 UDP 广播到设备发现端口
        DISCOVERY_PORT = 31337
        DISCOVERY_MSG = b"LIMA_DISCOVER"
        sock = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
        sock.setsockopt(socket.SOL_SOCKET, socket.SO_BROADCAST, 1)
        sock.settimeout(2.0)
        sock.sendto(DISCOVERY_MSG, ("255.255.255.255", DISCOVERY_PORT))

        devices = []
        try:
            while True:
                data, addr = sock.recvfrom(4096)
                info = json_mod.loads(data.decode())
                info["ip"] = addr[0]
                devices.append(info)
        except socket.timeout:
            pass
        finally:
            sock.close()

        # 过滤已绑定的设备
        with connect() as conn:
            bound_sns = {
                row["device_sn"] for row in conn.execute(
                    "SELECT device_sn FROM v2_device_binding WHERE status != 'unbound'"
                ).fetchall()
            }
        result = [
            {**d, "bound": d.get("deviceSn", "") in bound_sns}
            for d in devices
        ]
        return ok({"devices": result, "source": "server_scan"})
    except Exception as exc:
        return err(500, f"discovery failed: {exc}", 500)
```

```python
@router.post("/devices/pair/confirm")
async def confirm_pair(request: Request) -> JSONResponse:
    """设备配网确认（设备连上云端后调用，用配网令牌完成绑定）。

    此端点无需用户认证，使用配网令牌认证。
    """
    body = await read_body(request)
    if isinstance(body, JSONResponse):
        return body
    pair_token = str_field(body, "pairToken", "pair_token")
    device_sn = str_field(body, "deviceSn", "device_sn")
    if not pair_token or not device_sn:
        return err(400, "pairToken and deviceSn are required", 400)

    with connect() as conn:
        row = conn.execute(
            "SELECT * FROM v2_pair_request WHERE pair_token=? AND status='pending'",
            (pair_token,)
        ).fetchone()
        if not row:
            return err(404, "invalid or expired pair token", 404)

        # 检查是否过期
        import time
        if int(row["expires_at"]) < time.time():
            conn.execute(
                "UPDATE v2_pair_request SET status='expired' WHERE id=?", (row["id"],)
            )
            conn.commit()
            return err(410, "pair token expired", 410)

        # 完成绑定
        account_id = row["account_id"]
        conn.execute(
            """INSERT OR REPLACE INTO v2_device_binding
               (device_sn, account_id, status, bound_at)
               VALUES (?,?, 'bound', ?)""",
            (device_sn, account_id, now())
        )
        conn.execute(
            "UPDATE v2_pair_request SET status='completed' WHERE id=?", (row["id"],)
        )
        conn.commit()

    return ok({"deviceSn": device_sn, "status": "bound", "accountId": account_id})
```

```python
                COUNT(*) as total_tasks,

                SUM(CASE WHEN status='cancelled' THEN 1 ELSE 0 END) as cancelled_tasks,

            """SELECT capability, COUNT(*) as cnt
               FROM v2_task WHERE device_id=? AND created_at>=?
               GROUP BY capability ORDER BY cnt DESC LIMIT 1""",
            (device_id, start_str)
        ).fetchone()

        # 每日任务数
        daily_rows = conn.execute(
            """SELECT DATE(created_at) as date, COUNT(*) as count,
                      SUM(CASE WHEN status='completed' THEN 1 ELSE 0 END) as completed
               FROM v2_task WHERE device_id=? AND created_at>=?
               GROUP BY DATE(created_at) ORDER BY date""",
            (device_id, start_str)
        ).fetchall()

        # 每小时使用分布
        hourly_rows = conn.execute(
            """SELECT CAST(strftime('%H', created_at) AS INTEGER) as hour, COUNT(*) as count
               FROM v2_task WHERE device_id=? AND created_at>=?
               GROUP BY hour ORDER BY hour""",
            (device_id, start_str)
        ).fetchall()

    total = overview["total_tasks"] or 0
    completed = overview["completed_tasks"] or 0
    failed = overview["failed_tasks"] or 0

    hourly_pattern = [0] * 24
    for row in hourly_rows:
        hourly_pattern[row["hour"]] = row["count"]
```

```python
@router.post("/devices/provision")
async def provision_device(request: Request, authorization: str = Header(default="")) -> JSONResponse:
    """设备配网引导：生成 SoftAP 模式配网信息。

    此端点是 M9 pair 端点的简化版，适用于服务器端直接生成配网包
    而不需要小程序参与 UDP 发现流程的场景。
    """
    account = authorize(authorization)
    if isinstance(account, JSONResponse):
        return account
    body = await read_body(request)
    if isinstance(body, JSONResponse):
        return body
    device_sn = str_field(body, "deviceSn", "device_sn")
    wifi_ssid = str_field(body, "wifiSsid", "ssid")
    wifi_password = str_field(body, "wifiPassword", "password")
    if not device_sn or not wifi_ssid:
        return err(400, "deviceSn and wifiSsid are required", 400)

    # 生成一次性配网令牌（30分钟有效）
    import secrets, time
    provision_token = secrets.token_urlsafe(32)
    expires_at = time.time() + 1800

    # 存储配网请求
    provision_id = new_id()
    with connect() as conn:
        conn.execute(
            """INSERT INTO v2_pair_request
               (id, pair_token, device_sn, account_id, wifi_ssid, server_url,
                created_at, expires_at, status)
               VALUES (?,?,?,?,?,?,?,?,?)""",
            (provision_id, provision_token, device_sn, account["id"],
             wifi_ssid, f"wss://{request.url.hostname}/device/v1/ws",
             now(), str(int(expires_at)), "pending")
        )
        conn.commit()

    # 返回完整配网包（设备端可直接消费的 JSON）
    return ok({
        "provisionId": provision_id,
        "provisionToken": provision_token,
        "deviceSn": device_sn,
        "serverUrl": f"wss://{request.url.hostname}/device/v1/ws",
        "protocol": "lima-device-v1",
        "expiresIn": 1800,
        "configPayload": {
            "wifi_ssid": wifi_ssid,
            "wifi_password": wifi_password,
            "server_url": f"wss://{request.url.hostname}/device/v1/ws",
            "pair_token": provision_token,
            "device_sn": device_sn,
        },
    })

@router.post("/devices/provision/confirm")
async def confirm_provision(request: Request) -> JSONResponse:
    """设备配网确认（设备连上云端后调用）。

    与 M9 的 /devices/pair/confirm 共享同一逻辑，
    使用配网令牌认证，无需用户 token。
    """
    body = await read_body(request)
    if isinstance(body, JSONResponse):
        return body
    provision_token = str_field(body, "provisionToken", "pair_token")
    device_sn = str_field(body, "deviceSn", "device_sn")
    if not provision_token or not device_sn:
        return err(400, "provisionToken and deviceSn are required", 400)

    with connect() as conn:
        row = conn.execute(
            "SELECT * FROM v2_pair_request WHERE pair_token=? AND status='pending'",
            (provision_token,)
        ).fetchone()
        if not row:
            return err(404, "invalid or expired provision token", 404)

        import time
        if int(row["expires_at"]) < time.time():
            conn.execute(
                "UPDATE v2_pair_request SET status='expired' WHERE id=?", (row["id"],)
            )
            conn.commit()
            return err(410, "provision token expired", 410)

        account_id = row["account_id"]
        conn.execute(
            """INSERT OR REPLACE INTO v2_device_binding
               (device_sn, account_id, status, bound_at)
               VALUES (?,?, 'bound', ?)""",
            (device_sn, account_id, now())
        )
        conn.execute(
            "UPDATE v2_pair_request SET status='completed' WHERE id=?", (row["id"],)
        )
        conn.commit()

    return ok({"deviceSn": device_sn, "status": "bound", "accountId": account_id})
```

```python
# routing/semantic_router.py
from numpy import dot
from typing import Optional

class SemanticRouter:
    """毫秒级语义路由预筛层"""
    
    def __init__(self, encoder, routes: list[dict], threshold: float = 0.85):
        self.encoder = encoder  # 本地 embedding 模型
        self.threshold = threshold
        self.route_embeddings = self._encode_routes(routes)
    
    def _encode_routes(self, routes):
        """预计算路由描述的 embedding"""
        return {r["name"]: self.encoder(r["description"]) for r in routes}
    
    def classify(self, query: str) -> Optional[tuple[str, float]]:
        """返回 (route_name, confidence) 或 None"""
        query_emb = self.encoder(query)
        scores = {name: dot(query_emb, emb) for name, emb in self.route_embeddings.items()}
        best = max(scores, key=scores.get)
        if scores[best] >= self.threshold:
            return (best, scores[best])
        return None  # 低置信度 → 走 LLM 分类
```

**验收标准**：
- 语义路由层延迟 < 5ms（本地 embedding）
- 高频请求（如“画个圆”、“写个字”）命中率 > 60%
- LLM 分类器调用量降低 50%+

**实现参考**（基于 prompt-cache / GPTCache）：
```python
# cache/semantic_cache.py
import time, json
from numpy import dot
from dataclasses import dataclass

@dataclass
class CacheEntry:
    query: str
    embedding: list[float]
    response: dict
    timestamp: float
    hit_count: int = 0

class SemanticCache:
    """语义缓存：相似请求直接返回"""
    
    def __init__(self, encoder, ttl: int = 3600, \
                 similarity_threshold: float = 0.92, max_size: int = 10000):
        self.encoder = encoder
        self.ttl = ttl
        self.similarity_threshold = similarity_threshold
        self.max_size = max_size
        self._store: dict[str, CacheEntry] = {}  # key → entry
    
    def get(self, query: str) -> dict | None:
        """查找语义相似的缓存项"""
        query_emb = self.encoder(query)
        now = time.time()
        best_score = 0.0
        best_entry = None
        for entry in self._store.values():
            if now - entry.timestamp > self.ttl:
                continue
            score = dot(query_emb, entry.embedding)
            if score > best_score:
                best_score = score
                best_entry = entry
        
        if best_entry and best_score >= self.similarity_threshold:
            best_entry.hit_count += 1
            return best_entry.response
        return None
    
    def set(self, query: str, response: dict):
        """写入缓存"""
        if len(self._store) >= self.max_size:
            self._evict_lru()
        self._store[query] = CacheEntry(
            query=query,
            embedding=self.encoder(query),
            response=response,
            timestamp=time.time()
        )
    
    def _evict_lru(self):
        """LRU 淘汰"""
        oldest = min(self._store.values(), key=lambda e: e.hit_count)
        del self._store[oldest.query]
```

**验收标准**：
- 缓存命中率 > 30%（高频场景如设备控制）
- 缓存命中延迟 < 5ms
- TTL + LRU 淘汰正常工作
- 缓存数据可用 Redis 持久化（可选）
- 后端调用量降低 30%+

```python
# observability/tracer.py
import time, uuid
from contextlib import contextmanager

class PipelineTracer:
    """18 步管线全链路追踪"""
    
    def __init__(self):
        self.run_id = str(uuid.uuid4())
        self.steps: list[dict] = []
    
    @contextmanager
    def trace_step(self, step_name: str, **inputs):
        """追踪单步执行"""
        step_id = len(self.steps) + 1
        start = time.time()
        result = None
        error = None
        try:
            yield
            result = "success"
        except Exception as e:
            error = str(e)
            result = "error"
            raise
        finally:
            elapsed_ms = (time.time() - start) * 1000
            self.steps.append({
                "run_id": self.run_id,
                "step_id": step_id,
                "step_name": step_name,
                "inputs": inputs,
                "result": result,
                "error": error,
                "elapsed_ms": elapsed_ms,
                "timestamp": time.time(),
            })
    
    def export(self) -> dict:
        """导出追踪数据（兼容 LangSmith 格式）"""
        return {
            "run_id": self.run_id,
            "total_steps": len(self.steps),
            "total_ms": sum(s["elapsed_ms"] for s in self.steps),
            "steps": self.steps,
        }
```

**验收标准**：
- 18 步管线每步的输入/输出/耗时/状态全部记录
- 追踪数据可用 LangSmith 或自建 UI 可视化
- 支持按 run_id 查询完整执行链
- 追踪开销 < 2%（单步 < 0.1ms）

## A.9 提示词系统 — 模板管理、路由决策与结构化输出

LiMa 18 步管线的入口是提示词系统：`routing_classifier.py` 分类 → `speculative.py` 复杂度评估 → `context_pipeline/` 上下文构建 → `routing_engine.py` 后端选择 → `orchestrate.py` 编排 → 后端执行。提示词质量直接决定路由准确率和输出质量。以下项目覆盖提示词模板管理、自动优化、结构化输出、语义路由、测试评估和编排执行。
