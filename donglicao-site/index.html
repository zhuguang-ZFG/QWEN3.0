Developer API

面向开发者的开放接口

兼容 OpenAI 格式的 API，支持设备控制（REST + WebSocket）、内容生成、多模型路由。一行代码接入 LiMa 云端能力。

打开控制台
quick_start.py
# 一句话生成绘图指令
import openai

client = openai.OpenAI(
  base_url="https://chat.donglicao.com/v1",
  api_key="your-key"
)

response = client.chat.completions.create(
  model="lima",
  messages=[{"role": "user",
    "content": "画一只猫"}]
)
